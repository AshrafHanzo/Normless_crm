# ⚡ ADMIN SYSTEM QUICK REFERENCE

## 🎯 5-MINUTE SETUP

```
1. Deploy files (upload via FTP)
2. SSH into server
3. npm install --production
4. pm2 restart normless-crm
5. Go to: https://app.normless.store/admin
6. Login: admin / admin123
7. Create first operator
8. Done! 🎉
```

---

## 👥 CREATE USER IN 30 SECONDS

```
1. Click: 👮 Admin (left sidebar)
2. Click: ➕ Create New User
3. Fill: username, password, role
4. Check: permissions you want
5. Click: ✅ Create User
DONE!
```

---
